│ 维度             │ 权重   │ 验证结果                                   │

│ tools_ok         │ 25%    │ ✓ 有效。工具调用不当时降分                 │
└──────────────────┴────────┴──────────────────────────────────────────┘

得分影响因素:
1. 长度不足的罚分系数 (0.3 vs 0.5):
   - 从0.5改为0.3后，长度失败的分数影响从 -0.125 降低到 -0.075
   - 使得明显"太短"的输出得分显著下降
   - 例：simple_002原本0.88，若输出更短可能降至0.7-0.8

2. Edge cases特殊系数:
   - 空结果处理 (0.85x): 认可了正确的容错处理，但降分反映不理想的结果
   - 安全防护 (上限0.75): 虽然安全防护正确重要，但没有工具调用也有损失

3. 关键词缺失的影响:
   - 如果关键词只匹配50%，总分约降低15% (0.30 * 0.5)
   - 关键词是最强的评分因子
